Bienvenido al juego de carros mas sencillo y apasionante: Drag Race
para ejecutar el juego debes correr el siguiente codigo en la consola de linux:

python3 main.py

Para la calificacion de SOFKA usar por favor la version 1.0 del archivo. La version 1.1 busca cuantas veces ha estado un jugador en el podio efectivamente,
pero se termino en una hora posterior a la fecha de entrega del programa.
